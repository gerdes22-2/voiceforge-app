# VoiceForge AI Studio - Song Processing Pipeline

1. **Upload & Validation**: Check format, duration, corruption.
2. **Analysis**: Detect BPM, Key, Genre, Language.
3. **Stem Separation**: MelBand-Roformer or Demucs.
4. **Vocal Processing**: Pitch extraction, Voice conversion (via Plugin).
5. **Mixing & Mastering**: Apply EQ, Compression, Reverb, Normalization.
6. **Export**: Convert to requested format.
